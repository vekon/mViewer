A simple web-based Administration and Management Tool for MongoDB.

mViewer(Micro/Mongo Viewer) is a light web-based GUI which manages MongoDB without needing any installation. mViewer allows managing databases, collections, gridfs, users & indexes as an alternative to the MongoDB native shell in simpler and user friendly way

Easy to login

Simple to login with and with out auth mode along with users created.

Easy to navigate

Easy to navigate betweeen databases and objects of database

Users creation & management

Easy to manage users. Create users and manage them with specific roles on selected database

Features

MULTIPLE DATABASES

Login & manage multiple mongo database instance hosted with different IP & Port.

AUTHENTICATION

Enhanced authentication. Works with and with-out auth mode.

USERS, ROLES & PRIVILEGES

Manage users by creating & assigning roles on different databases.

LOOK & FEEL

Good user interface with enhanced ease of use.

NAVIGATION

Ease in navigation b/w databases, collections & users of a database.

SUPPORTS MONGO 3.X

Supports latest mongodb driver 3.x.
